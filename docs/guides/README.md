# Guides d’utilisation – Baibebalo

Ces guides sont destinés à **former les utilisateurs** de la plateforme Baibebalo (clients, restaurants, livreurs) et à fournir une référence rapide au quotidien.

---

## Liste des guides

| Guide | Public | Contenu principal |
|-------|--------|-------------------|
| **[00 – Guide général](00-GUIDE-GENERAL.md)** | Tous | Vue d’ensemble de la plateforme, rôles, parcours d’une commande, bonnes pratiques communes. |
| **[01 – Guide Restaurant](01-GUIDE-RESTAURANT.md)** | Restaurants (gérants, cuisine) | Connexion, tableau de bord, **gestion des commandes** (accepter/refuser, détail, chat), **menu** (catégories, plats, promotions), statistiques, finances, paramètres, dépannage. |
| **[02 – Guide Livreur](02-GUIDE-LIVREUR.md)** | Livreurs | Inscription, statut (disponible/pause), **déroulement d’une course** (restaurant → client), vérification, code de confirmation, photo de preuve, **gains et remises espèces**, incidents (adresse incorrecte, client absent, urgence), stats, paramètres. |
| **[03 – Guide Client](03-GUIDE-CLIENT.md)** | Clients | Inscription, **commander** (restaurant, panier, adresse, paiement), **suivi de commande**, livraison et code de confirmation, avis, adresses, support. |

---

## Utilisation recommandée

- **Formation nouveaux partenaires (restaurants)** : lire le [Guide général](00-GUIDE-GENERAL.md) puis le [Guide Restaurant](01-GUIDE-RESTAURANT.md). Insister sur les sections « Commandes » et « Bonnes pratiques ».
- **Formation nouveaux livreurs** : lire le [Guide général](00-GUIDE-GENERAL.md) puis le [Guide Livreur](02-GUIDE-LIVREUR.md). Insister sur le « Déroulement d’une course », « Gestion des incidents » et « Gains / Remise espèces ».
- **Clients** : le [Guide Client](03-GUIDE-CLIENT.md) peut être mis à disposition dans l’app (aide) ou sur le site / support.
- **Support / FAQ** : les sections « Dépannage rapide » de chaque guide peuvent servir de base à une FAQ interne.

---

## Mise à jour des guides

Les libellés et écrans peuvent varier selon les versions des applications. Pensez à mettre à jour ces guides lors des évolutions majeures (nouveaux écrans, nouveaux flux, changement de termes).
