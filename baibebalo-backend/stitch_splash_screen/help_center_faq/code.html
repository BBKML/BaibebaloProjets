Questions fréquentes

Où est ma commande ?

expand_more

Vous pouvez suivre votre livreur en temps réel sur la carte depuis l'onglet 'Mes Commandes'. Le délai moyen à Abidjan est de 30-45 minutes.

Comment modifier mon mode de paiement ?

expand_more

Allez dans 'Compte' > 'Modes de paiement' pour ajouter une carte ou configurer votre portefeuille Mobile Money.

Quels sont les délais de livraison à Abidjan ?

expand_more

Nous livrons dans toutes les communes d'Abidjan. Les délais varient entre 20 min (Cocody, Plateau) et 60 min selon le trafic.
